Darwinbox session cookie
Required for downloading reports from Darwinbox links
How to refresh cookie (takes 2 minutes):
1. Open Chrome → cars24.darwinbox.com (stay logged in)
2. Press F12 → Application tab → Cookies → cars24.darwinbox.com
3. Find cookie named session → copy the Value
4. Open Apps Script → find SET_DARWINBOX_COOKIE() → paste value → Run
